# 2025년까지 윤리적 해킹 및 방어를 마스터하기 위한 70가지 실습형 사이버 보안 프로젝트

2025년에는 사이버 보안 프로젝트가  윤리적 해킹, 침투 테스트, 방어적 보안 분야에 진출하거나 경력을 쌓고자 하는 모든 사람에게 핵심적인 경험이 될 것입니다. 최근 몇 년 동안 랜섬웨어 공격이 50% 이상 증가하고 AI 기반 익스플로잇이 늘어나는 등 사이버 위협이 빠르게 진화함에 따라, 실질적인 사이버 보안 프로젝트를 통한 경험은 뛰어난 포트폴리오를 구축하는 데 그 어느 때보다 중요합니다. 초보자를 위한 사이버 보안 프로젝트, 중급 윤리적 해킹 랩, 고급 제로 트러스트 구현 등 어떤 프로젝트를 찾고 있든, 이 70가지 무료 랩 프로젝트 목록은 기초 개념부터 기업 수준의 방어 체계 구축까지 여러분을 안내할 것입니다.

GitHub에 진행 상황을 기록하고, 상세한 보고서를 작성하며, 윤리적인 방식으로 과정을 공유하여 사이버 보안 생태계에 기여하십시오.

## 2025년에 실무 중심의 사이버 보안 프로젝트에 시간을 투자해야 하는 이유는 무엇일까요?

실무 중심의 사이버 보안 프로젝트는 이론과 실무의 간극을 메워주며, Nmap, Wireshark, Hashcat과 같은 도구와 최신 AI 프레임워크를 숙달하는 데 도움을 줄 뿐만 아니라 CompTIA Security+, CEH, OSCP, CISSP와 같은 자격증 취득에도 기여합니다. 고용주들은 취약점 평가, 침입 탐지, 위협 탐색 분야에서 검증된 역량을 갖춘 지원자를 선호하며, 프로젝트는 이러한 역량을 입증하는 실질적인 증거를 제공합니다.

사이버 보안 프로젝트 포트폴리오를 구축하면 취업 경쟁력을 높일 수 있습니다. GitHub 저장소에 코드, 스크린샷, 프로젝트 설명 등을 링크드인에 공유하세요. 전 세계적으로 50만 개 이상의 사이버 보안 관련 일자리가 공석인 상황에서, 이러한 윤리적 해킹 프로젝트와 자체 연구실 구축 경험은 주도성과 문제 해결 능력을 보여주는 좋은 지표가 될 수 있습니다.

2025년 트렌드를 앞서나가세요: AI 기반 위협, 양자 컴퓨팅에 대한 내성 암호화, 클라우드 보안 상태 관리, IoT 취약점 등에 대비하려면 실무 경험이 필수적입니다. 작은 것부터 시작하여 꾸준히 발전해 나가면 자신감과 경력이 함께 성장할 것입니다.

## 초보자용 사이버 보안 프로젝트(1~25): 안전하게 기본기를 익히세요

이 초보자용 사이버 보안 프로젝트들은 2025년에 윤리적 해킹 여정을 시작하는 모든 사람에게 적합합니다. 각 프로젝트는 2~8시간 정도 소요되며, 노트북과 무료 도구(Kali Linux, VirtualBox, Windows VM)만 있으면 되고, 격리된 환경에서 수행하면 100% 합법적이고 윤리적입니다. 공격 원리를 이해하는 데 집중하여 더욱 강력한 방어 체계를 구축해 보세요.

- [공격자를 잡기 위한 기본적인 허니팟 설정하기](https://siemtune.com/how-to-set-up-honeypot-cybersecurity/)  가상 머신(VM) 내부에 Cowrie SSH/Telnet 허니팟을 배포하고 스크립트를 실행하는 공격자가 기본 자격 증명을 사용하여 실시간으로 시도하는 것을 관찰하세요. 기만 기술과 로그 분석을 처음 접하는 분들에게 적합합니다.[Cowrie GitHub 및 문서 보기](https://github.com/cowrie/cowrie)

- [간단한 암호 크래커를 만들어 보세요.](https://siemtune.com/password-cracking-tutorial-build-tools-for-ethical-hacking/)  파이썬 스크립트를 작성하거나 John the Ripper/Hashcat을 사용하여 직접 생성한 MD5, NTLM 또는 SHA-256 해시를 크랙해 보세요. 사전 공격, 하이브리드 공격, 마스크 공격을 배우고 강력하고 솔트가 포함된 암호가 왜 중요한지 알아보세요. 교육 목적으로만 사용하세요.[Hashcat 공식 웹사이트](https://hashcat.net/hashcat/)

- [네트워크 트래픽을 분석하는 도구를 만들어 보세요.](https://siemtune.com/build-custom-packet-sniffer-tutorial-scapy-lab/)  Scapy 또는 libpcap을 사용하여 사용자 지정 패킷 스니퍼를 작성하고 랩 네트워크에서 HTTP, DNS 및 암호화되지 않은 자격 증명을 캡처해 보세요. TLS 1.3이 협상 불가능한 이유를 즉시 확인할 수 있습니다.[Scapy 문서 참조](https://scapy.net/)

- 연습용 기본 키로거를 만들어 보세요.  파일에 기록하는 Python 키로거(VM에서만 실행)를 개념 증명용으로 제작합니다. Process Explorer와 안티바이러스 프로그램을 사용하여 탐지 방법을 연구하세요. 목표는 악성코드의 동작 방식을 학습하는 것이지 배포 방법을 학습하는 것이 아닙니다. [MITRE ATT&CK – 입력 캡처 T1056](https://attack.mitre.org/techniques/T1056/)

- 디지털 포렌식 분석 기술을 익히고  , Autopsy에서 샘플 포렌식 이미지를 마운트하여 삭제된 파일, 브라우저 기록, USB 흔적을 복구해 보세요. 사고 대응 워크플로에 대한 완벽한 입문 과정입니다. [Autopsy 공식 웹사이트를 방문하세요.](https://www.autopsy.com/)

- VirtualBox 또는 VMware Workstation Player를 사용하여 Kali 공격 환경, Windows 10/11 피해자 환경, Metasploitable 2 및 pfSense 방화벽으로 격리된 네트워크를 구축하여 집에서 개인 사이버 보안 연구실을 만들어 보세요 . 이는 모든 미래 프로젝트의 기반이 될 것입니다. [VirtualBox 설명서를 참조하세요](https://www.virtualbox.org/manual/)  .

- 암호화를 사용하여 파일을 암호화하는 연습을 하세요.  GPG 또는 OpenSSL을 사용하여 AES-256 및 RSA-4096으로 파일을 암호화/복호화해 보세요. 하이브리드 암호화 및 암호 강도를 다양하게 실험해 보세요. [GnuPG 문서를 참조하세요 .](https://gnupg.org/documentation/)

- 피싱 캠페인 시뮬레이션을 통해 경각심을 고취하세요.  Gophish를 사용하여 동의를 얻은 후 피싱 시뮬레이션을 실행해 보세요. 이메일을 작성하고, 가짜 랜딩 페이지를 호스팅하고, 클릭을 추적할 수 있습니다. 개인 또는 팀 교육에 매우 유용합니다. [Gophish는 오픈 소스 소프트웨어입니다.](https://getgophish.com/)

- Aircrack-ng 제품군을 사용하여 집에서 Wi-Fi 보안을 테스트하고  , 네트워크에서 WPA2/3 핸드셰이크를 캡처하여 오프라인 크래킹을 시도해 보세요. 강력하고 고유한 암호와 WPA3의 필요성을 다시 한번 확인해 보세요. [Aircrack-ng 공식 웹사이트를 이용해 보세요.](https://www.aircrack-ng.org/)

- 네트워크 취약점을 스캔하세요.  Metasploitable 3 또는 랩 VM에서 Nmap NSE 스크립트와 OpenVAS를 실행하세요. 전문적인 취약점 보고서를 생성하고 해결 우선순위를 파악하세요. [Nmap.org](https://nmap.org/)

- 위협을 차단하기 위한 방화벽 규칙 구성 합법적인 서비스를 허용하면서 악성 트래픽을 삭제하려면 정확한 iptables, nftables, UFW 또는 Windows 방화벽 규칙을 작성하세요. hping3 및 nmap으로 테스트합니다. [우분투 UFW 가이드](https://help.ubuntu.com/community/UFW)

- 2단계 인증 시스템 생성 pyotp를 사용하여 Python에서 TOTP 인증자를 구축하고 이를 간단한 Flask 로그인 페이지에 통합합니다. Google Authenticator 이외의 시간 기반 일회용 비밀번호를 이해하세요. [PyPI의 pyotp](https://pypi.org/project/pyotp/)

- 보안 웹 애플리케이션 구축 OWASP 권장 사항을 사용하여 SQL 주입, XSS, CSRF 및 클릭재킹을 처음부터 방지하는 Flask 또는 Django todo 앱을 개발합니다. [OWASP 보안 코딩 관행](https://owasp.org/www-project-secure-coding-practices-quick-reference-guide/)

- 침입 탐지를 위해 Snort를 설정하고 사용합니다. Emerging Threats 규칙이 포함된 Snort 3를 설치하고 실험실에서 시뮬레이션된 공격(nmap 스캔, Metasploit 익스플로잇)에 대한 경고를 생성합니다. [Snort.org](https://www.snort.org/)

- 도구를 사용하여 시스템 취약점을 찾습니다. 취약한 VM에 대해 Lynis, OpenVAS 또는 Nikto를 실행하고 CVE, 잘못된 구성 및 약한 권한을 해석합니다. [CISofy 린스](https://cisofy.com/lynis/)

- DNS 스푸핑 공격 탐지 연습 Ettercap 또는 dnsspoof를 사용하여 ARP + DNS 스푸핑을 시뮬레이션하고 DNSSEC, HTTPS-Everywhere 및 엄격한 확인자 설정으로 방어합니다. [Ettercap 프로젝트](https://www.ettercap-project.org/)

- 바이러스 백신 소프트웨어가 어떻게 작동하는지 연구합니다. EICAR 테스트 파일 및 안전한 맬웨어 샘플에 대해 ClamAV, Windows Defender 및 오픈 소스 서명을 테스트합니다. 휴리스틱과 서명을 알아보세요. [ClamAV.net](https://www.clamav.net/)

- 비정상적인 활동을 감지하는 도구를 구축합니다. 프로세스 생성, 파일 변경 및 네트워크 연결을 모니터링하는 Python의 간단한 호스트 기반 이상 감지기를 스크립트합니다. [비트 트레일별 OSQuery](https://osquery.io/)

- 맬웨어를 분석하여 동작을 이해합니다. VirusTotal 샘플에 대한 안전한 정적 분석을 수행하고 PEiD, Strings 및 ProcMon과 같은 도구를 사용하여 REMnux VM에서 동적 분석을 수행합니다. [REMnux 문서](https://remnux.org/)

- 보안 통신을 위해 TLS를 설정하세요. Let’s Encrypt에서 무료 인증서를 받고 완벽한 전달 보안 및 HSTS 헤더로 Nginx/Apache를 구성하세요. [암호화 시작하기](https://letsencrypt.org/getting-started/)

- 제로데이 익스플로잇의 작동 방식 알아보기 Google Project Zero 및 Microsoft의 공개 제로데이 공개(익스플로잇 코드 없음)를 연구하여 취약성 클래스 및 책임 공개를 이해합니다. [프로젝트 제로 블로그](https://googleprojectzero.blogspot.com/)

- 개인정보 보호를 위해 Tor와 같은 익명 도구를 사용하세요. Tor 브라우저, 양파 서비스 및 브리지를 구성하세요. 운영 보안과 익명성 절충점을 살펴보세요. [토르 프로젝트](https://www.torproject.org/)

- 위협 데이터를 수집하기 위한 허니팟 설정 20개 이상의 허니팟이 포함된 올인원 허니팟 플랫폼인 T-Pot을 배포하고 시각화를 위해 로그를 ELK에 전달합니다. [GitHub의 T-Pot](https://github.com/telekom-security/tpotce)

- GPU로 비밀번호 크래킹 연습 NVIDIA/AMD GPU와 함께 Hashcat을 사용하여 마스크 공격을 벤치마킹하고 기호가 포함된 12자 이상의 비밀번호 문구가 최신 하드웨어를 무력화하는 이유를 보여줍니다. [Hashcat 위키 예](https://hashcat.net/wiki/)

- 보안을 위한 브라우저 확장 프로그램 만들기 알려진 피싱 도메인을 차단하고 HTTPS를 시행하거나 추적 매개변수를 제거하는 Chrome/Firefox 확장 프로그램을 만듭니다. [Mozilla 웹 확장](https://developer.mozilla.org/en-US/docs/Mozilla/Add-ons/WebExtensions)

## 중급 사이버 보안 프로젝트(26~50): 맞춤형 도구, 자동화 및 AI 기반 방어 구축

이러한 중급 사이버 보안 프로젝트는 실제 공격 및 방어 도구를 사용하도록 유도합니다. 프로젝트당 6~15시간, Python 또는 Bash의 더 무거운 스크립팅, 고급 가상화, 보안에 적용되는 머신러닝의 첫 경험을 기대하세요. 모든 빌드는 100% 실험실에 포함되어 있고 윤리적입니다. 2025년 침투 테스트 또는 블루팀 포트폴리오를 강화하는 데 적합합니다.

- 악성코드 분석 샌드박스 구축
Cuckoo Sandbox 또는 VirtualBox/Proxmox 내부의 맞춤형 REMnux 워크스테이션을 사용하여 완전히 자동화되고 격리된 악성 코드 제거 환경을 설정하세요. 정적/동적 분석 워크플로우, INetSim을 통한 네트워크 모니터링, 동작 보고서 생성에 대해 알아보고 프로덕션 시스템에서 라이브 악성코드를 멀리 유지하세요.
[쿠쿠 샌드박스 문서](https://cuckoo.readthedocs.io/en/latest/)

- 전체 디스크 암호화로 데이터 암호화
Windows/Linux에서 VeraCrypt 숨겨진 볼륨을 구현하거나 Ubuntu/Rocky Linux에서 LUKS + dm-crypt를 구현하세요. 그럴듯한 거부성, 키 파일 사용 및 긴급 분리 절차를 테스트합니다. 이는 레드팀 참여 또는 개인 정보 보호 프로젝트 중에 저장되어 있는 민감한 데이터를 보호하는 데 필수적인 기술입니다.
[VeraCrypt 공식 가이드](https://www.veracrypt.fr/en/Documentation.html)

- 침입 탐지를 위해 기계 학습을 사용합니다.
CIC-IDS2017/2018 또는 NSL-KDD 데이터 세트를 사용하여 scikit-learn 또는 TensorFlow로 지도형 침입 탐지 모델을 훈련합니다. Zeek 또는 Suricata 로그를 처리하고 이상 징후에 대한 경고를 처리하는 실시간 Python 파이프라인을 구축하세요. 이는 2025년에 처음부터 새로 구축된 최초의 기계 학습 침입 탐지 시스템입니다.
[캐나다 사이버 보안 데이터세트](https://www.unb.ca/cic/datasets/)

- 보안 라우팅 프로토콜 테스트
GNS3 또는 EVE-NG에서 다중 라우터 랩을 가동하고 BGP 경로 하이재킹을 시뮬레이션한 다음 RPKI, 경로 필터링 및 IRR 검증으로 방어합니다. OSPF MD5 인증을 추가하고 안전하지 않은 프로토콜에 대한 중간자 공격을 테스트합니다.
[Cisco BGP 보안 모범 사례](https://www.cisco.com/)

- 안전한 암호화폐 지갑 만들기
Electrum을 사용하여 에어갭 계층적 결정론적 지갑을 생성하거나 python-bitcoinlib 및 BIP-39/44 표준을 사용하여 직접 코딩하세요. 2025년 암호화폐 자산 보호를 이해하는 데 중요한 시드 백업, 다중 서명 설정, 거래 서명을 오프라인으로 연습해 보세요.
[Electrum 문서](https://electrum.readthedocs.io/en/latest/)

- 루트킷 및 제거 방법 연구
Volatility 3, chkrootkit 및 rkhunter를 사용하여 오픈 소스 사용자 영역 및 커널 루트킷(방어에만 집중)을 분석합니다. 손상된 랩 VM에 대한 실시간 대응 기술 및 커널 모듈 검사를 연습합니다.
[Volatility 3 프레임워크](https://volatility-labs.blogspot.com/)

- 연구용 다크 웹 스크레이퍼 구축
Stem 및 BeautifulSoup을 사용하여 Python으로 윤리적인 Tor 인식 스크레이퍼를 작성하여 위협 인텔리전스 연구를 위해 공개적으로 사용 가능한 양파 서비스 데이터를 수집합니다. Tor 양말 프록시를 통해 모든 것을 라우팅하고 robots.txt를 존중합니다. 합법적인 OSINT 수집만 가능합니다.
[Tor 프로젝트 줄기 라이브러리](https://stem.torproject.org/)

- 안전한 환경에서 DDoS 공격을 시뮬레이션합니다.
자체 서비스에 대해 hping3, SlowHTTPTest 및 LOIC와 같은 안전한 저속 도구를 사용하여 볼륨 측정, 프로토콜 및 애플리케이션 계층 DDoS 벡터에 대한 개념적 심층 분석을 수행합니다. 속도 제한 및 Cloudflare 스타일 과제를 통해 탐지 및 완화에 집중하세요.
[Cloudflare DDoS 학습 센터](https://www.cloudflare.com/learning/ddos/)

Signal Protocol(libsignal-protocol-python) 또는 Noise Protocol 프레임워크를 사용하여 종단 간 암호화된 채팅 애플리케이션의 프로토타입을 만듭니다. 순방향 보안, 키 래칫 및 오프라인 메시지 암호화를 구현합니다.

- PKI(공개 키 인프라) 구축
OpenSSL 또는 EJBCA를 사용하여 자체 루트 및 중간 CA를 생성하고, 클라이언트/서버 인증서를 발급하고, OCSP 응답자를 설정하고, CRL을 사용하여 취소를 시행합니다. 내부 랩에서 사용할 수 있는 개인 인증 기관입니다.
[OpenSSL CA 튜토리얼](https://jamielinux.com/docs/openssl-certificate-authority/)

- 제로데이 취약점 연구
Microsoft, Google Project Zero 및 Apple의 최근 책임 공개, 버그 포상금 프로그램, 패치 타임라인을 살펴보세요. 익스플로잇 코드를 작성하지 않고 익스플로잇 기본 요소를 연구합니다.
[구글 프로젝트 제로 블로그](https://googleprojectzero.blogspot.com/)

- 보안 문제에 대한 스마트 계약 감사

[ConsenSys MythX](https://mythx.io/)

- 보안 테스트를 위한 맞춤형 도구 구축
자체 정찰 또는 악용 후 유틸리티를 개발하십시오. DNSSEC 지원, 맞춤형 SMB 악용 모듈 또는 피싱 키트용 자동 스크린샷을 지원하는 고급 하위 도메인 무차별 공격을 생각해 보세요.
[Rapid7 Metasploit 프레임워크](https://github.com/rapid7/metasploit-framework)

- AI를 활용한 위협 탐지
시스템 호출 추적 및 네트워크 흐름에서 Isolation Forest, 자동 인코더 또는 LSTM 네트워크를 사용하여 비지도 이상 탐지를 생성합니다. 이는 제로데이 기존 서명 누락을 찾아내는 데 적합합니다.
[scikit-learn 이상 탐지](https://scikit-learn.org/stable/modules/outlier_detection.html)

- 결함을 찾기 위한 리버스 엔지니어링 펌웨어
binwalk 및 Ghidra(NSA의 오픈 소스 리버스 엔지니어링 도구)를 사용하여 IoT/라우터 펌웨어를 추출하고 분해합니다. 항상 합법적으로 획득한 이미지에서 하드 코딩된 자격 증명, 안전하지 않은 업데이트 메커니즘, 버퍼 오버플로를 찾아보세요.
[기드라 공식](https://ghidra-sre.org/)

[OWASP IoT 프로젝트](https://owasp.org/www-project-internet-of-things/)

Terraform, Ansible 및DetectionLab을 사용하여 현실적인 레드팀/블루팀 연습을 스크립트로 작성하세요. 랜섬웨어 시뮬레이션, 측면 이동 경로, 측정 가능한 탐지 목표를 포함합니다.
[DetectionLab GitHub](https://github.com/clong/DetectionLab)

[실용 악성코드 분석 도서 자료](https://nostarch.com/malware)

Rekall 또는 WinDbg를 사용하여 Credential Guard, LSA 보호, AMSI 우회 기술(탐지 측면) 및 메모리 포렌식을 살펴보세요.

안전한 오픈 소스 사용자 영역 루트킷 예제(교육용)가 포함된 개념 가이드와 최신 EDR 우회 연구를 사용한 탐지/제거에 중점을 두고 있습니다. 이는 엄격한 방어 지향입니다.

## 고급 수준 사이버 보안 프로젝트(51-70): 엔터프라이즈, 클라우드, AI 및 신흥 기술

이러한 캡스톤 프로젝트는 2026년 이후를 준비하는 고위 실무자, DevSecOps 엔지니어, 위협 사냥꾼 및 연구원을 위해 구축되었습니다. 프로젝트당 15~30시간 이상, 철저한 자동화, 클라우드 기반 도구 사용, AI/ML 모델 통합이 예상됩니다. 모든 빌드는 격리된 연구실, 테스트 테넌트 또는 승인된 환경을 위해 설계되었으며 명시적인 승인 없이는 절대 생산되지 않습니다.

- 클라우드 보안 시스템 모니터링 AWS GuardDuty, Azure Sentinel 및 GCP Security Command Center를 사용하여 다중 클라우드 보안 모니터링 파이프라인을 배포한 다음 SOAR을 통해 자동 응답 플레이북을 사용하여 결과를 중앙 [SIEM](https://siemtune.com/what-is-a-siem/)(Elastic 또는 Splunk 무료 계층)으로 전달합니다. [AWS Security Hub 설명서](https://docs.aws.amazon.com/securityhub/)

- 제로 트러스트 보안 모델 구축 Cloudflare Access, Google BeyondCorp Enterprise 또는 위치에 관계없이 모든 요청을 확인하는 오픈 소스 대안(SPIFFE/SPIRE + OPA)을 사용하여 완전한 BeyondCorp 스타일 제로 트러스트 아키텍처를 구현합니다. [NIST SP 800-207 제로 트러스트 아키텍처](https://nvlpubs.nist.gov/nistpubs/SpecialPublications/NIST.SP.800-207.pdf)

- 랜섬웨어 탐지 도구 생성 Sysmon + Osquery 로그, Elastic EQL 규칙, 파일 엔트로피 스파이크 및 랜섬 노트 패턴에 대해 훈련된 경량 ML 모델을 결합한 행동형 랜섬웨어 조기 경고 시스템을 개발합니다. [Elastic Security 랜섬웨어 예방](https://www.elastic.co/security)

- AI 모델로 사이버 위협 예측 공개 위협 인텔 피드(MISP, AlienVault OTX) 및 과거 공격 데이터를 기반으로 LSTM 또는 Prophet을 사용하여 위협 예측 엔진을 구축하여 부문 및 지역별 캠페인 가능성을 예측합니다. [MITRE ENGAGE 예측 분석](https://engage.mitre.org/)

- 소셜 미디어의 사이버 보안 위험 연구 Maltego 및 맞춤형 API 스크래퍼를 사용하여 X, TikTok 및 Meta 플랫폼 전반에 걸쳐 프로파일링, 잘못된 정보 확산, 딥페이크 탐지 및 계정 탈취 벡터를 위한 포괄적인 OSINT 연구 프레임워크입니다. [Bellingcat 소셜 미디어 OSINT 가이드](https://www.bellingcat.com/resources/)

- 로그용 Splunk를 사용하여 대시보드 구축 클라우드, 엔드포인트 및 네트워크 로그를 Splunk Free/Developer 버전으로 수집하고 상관 검색, 위험 기반 경고 및 UEBA를 갖춘 실시간 보안 운영 센터 대시보드를 생성합니다. [Splunk 개발자 포털](https://developer.splunk.com/)

- 새로운 위협 인텔리전스 방법 연구 오픈 소스 MISP + OpenCTI를 사용하여 STIX/TAXII 2.1 피드를 자동화하고, 생성 AI 요약(로컬 LLM)을 강화하고, 자동화된 일일 인텔리전스 브리핑을 생성합니다. [MISP 프로젝트](https://www.misp-project.org/)

- 양자 암호화 기본 사항 탐색 NIST PQC 표준화 4차에서 포스트퀀텀 알고리즘(Kyber, Dilithium)과 양자 위협 범위에 대비한 프로토타입 하이브리드 인증서 체인을 연구합니다. [NIST 포스트 양자 암호화](https://csrc.nist.gov/projects/post-quantum-cryptography)

- 실제 APT 샘플(격리된 샌드박스에서 안전하게 폭발)에 대한 고급 Ghidra + IDA Python 스크립팅을 위한 리버스 엔지니어링 악성코드로 IoC를 추출하고 C2 구성을 해독하며 ​​YARA/Sigma 탐지 규칙을 생성합니다. [NSA 기드라](https://ghidra-sre.org/)

- 스마트 홈 장치의 취약점 테스트 Zigbee, Z-Wave 및 Matter 장치에 대한 전체 장치 평가를 수행합니다. KillerBee를 사용한 무선 스니핑, 펌웨어 추출 및 제조업체에 대한 책임 있는 결과 공개. [OWASP IoT 보안 프로젝트](https://owasp.org/www-project-internet-of-things/)

- 인증 도구를 사용한 보안 API OAuth 2.1/OIDC, mTLS 및 Kong Gateway 또는 Envoy가 시행하는 세분화된 ABAC 정책을 사용하여 Keycloak 또는 Authentik을 엔터프라이즈 IdP로 배포합니다. [키클로크 문서](https://www.keycloak.org/documentation)

- 레드 및 블루 팀 연습 연습 MITRE Caldera 또는 Covenant를 사용하여 DetectionLab 환경에 대해 완전한 적 에뮬레이션을 실행하고, 탐지 범위를 측정하고, 퍼플 팀 플레이북을 조정하세요. [칼데라 GitHub](https://github.com/mitre/caldera)

- 블록체인 시스템의 취약점 찾기 Foundry 및 공식 검증 도구를 사용하여 레이어 1 및 레이어 2 프로토콜과 51% 공격, MEV, 브리지 익스플로잇 및 합의 버그에 대한 DeFi 스마트 계약을 감사합니다. [비트 블록체인 보안의 흔적](https://github.com/crytic/building-secure-contracts)

- SCADA 시스템의 보안 연구 MiniCPS 또는 Dragos 개발 도구를 사용하여 ICS 환경을 시뮬레이션하고 IEC 62443 영역 분할, 프로토콜 이상 탐지 및 수동 자산 검색을 실행합니다. [NIST SP 800-82r3 ICS 보안](https://nvlpubs.nist.gov/nistpubs/SpecialPublications/NIST.SP.800-82r3.pdf)

- 간단한 생체 인식 인증 시스템 구축 OpenCV + 로컬 차등 개인 정보 보호 또는 보안 엔클레이브(Intel SGX/ARM TrustZone)를 사용한 얼굴/음성 인식을 사용하여 개인 정보 보호 생체 인식 로그인 프로토타입을 만듭니다. [OpenCV 얼굴 인식 튜토리얼](https://opencv.org/)

- 시스템에 대한 취약성 검색을 자동화합니다. Jira/ServiceNow의 자동 티켓 검색 결과인 Ansible 또는 Terraform Cloud를 사용하여 클라우드 및 온프레미스 자산 전반에 걸쳐 Nessus/OpenVAS + Nuclei 검색을 조율합니다. [테너블 Nessus API](https://developer.tenable.com/docs/welcome)

- 마이크로서비스로 컨테이너를 보호합니다. Pod 보안 승인, Falco 런타임 감지, Kyverno 정책 및 eBPF 기반 네트워크 세분화를 통해 Kubernetes 클러스터를 강화합니다. [Kubernetes 강화 가이드 CNCF](https://kubernetes.io/docs/concepts/security/)

- 자율주행차 사이버보안 프로젝트 연구 오픈 소스 도구와 ISO/SAE 21434 규정 준수 프레임워크를 사용하여 CAN 버스, V2X(DSRC/C-V2X) 및 무선 업데이트 공격을 분석합니다. [SAE J3061 사이버 보안 가이드북](https://www.sae.org/standards/content/j3061_202112/)

- 스테가노그래피 도구를 사용하여 숨겨진 데이터 감지 StegExpose, BossBase 데이터 세트에 대해 훈련된 사용자 정의 CNN 모델, 비밀 채널에 대한 포렌식 오디오/비디오 검사를 사용하여 스테가나리시스 파이프라인을 구축합니다. [StegExpose GitHub](https://github.com/b3dk7/StegExpose)

- 사이버 보안 교육 프로그램을 개발하고  , CTF 이벤트, 피싱 시뮬레이션, 보안 코딩 부트캠프, NIST 800-50에 부합하는 측정 가능한 인식 지표를 포함하는 완벽한 사내 프로그램을 설계 및 제공합니다. [NIST SP 800-50 보안 인식 교육](https://nvlpubs.nist.gov/nistpubs/Legacy/SP/nistspecialpublication800-50.pdf)
